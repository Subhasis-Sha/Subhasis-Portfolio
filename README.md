# My Portfolio

A modern and responsive personal portfolio website showcasing my projects, technical skills, education, and professional journey as a B.Tech student and aspiring web developer.

## Features

* Responsive design for desktop and mobile devices
* Modern and user-friendly interface
* About Me section
* Skills showcase
* Projects portfolio
* Contact information
* Clean and organized code structure

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

## Project Structure

```text
My-Portfolio/
├── images/
├── js/
├── static/
├── templates/
└── README.md
```

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/Subhasis-Sha/My-Portfolio.git
   ```

2. Open the project folder.

3. Launch `index.html` in your browser.
